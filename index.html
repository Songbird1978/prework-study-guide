Prework Study Guide

HTML

  • The head element contains information about the webpage. This includes elements such as metadata, styles used, descriptions of the page and authors.
  • The body element represents the visible content shown to the user, including text and images.
  • The body element uses sections elements to separate information.
  • The body element uses section elements to separate and organise information.
  • Headings use h1 - h6, from most important to least important.
  • Inside the body element (sections), You can list information using {li}
  • You can create numbered list elements inside {ol} and bullet points inside {ul} commands
